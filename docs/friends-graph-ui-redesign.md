# 朋友关系图谱 UI 重构说明

## 变更概述

将"朋友"分类下的关系图谱从开发态工具风格改造为符合博客极简主题的"朋友关系地图卡片"。

## 修改的文件

1. `src/components/links/friends-graph/FriendsGraph.astro` - 完全重构 UI 结构和样式
2. `src/components/links/friends-graph/friends-graph-client.ts` - 更新交互逻辑以支持新 UI

## 主要变更

### 1. 整体结构改造

**之前：** 图谱 + 外挂设置栏 + 外挂详情栏（三栏布局）

**现在：** 单一卡片包含所有元素

```
┌─────────────────────────────────────┐
│ 卡片头部（标题 + 工具按钮）          │
├─────────────────────────────────────┤
│ 图谱画布（带图例）                   │
│   └─ 详情面板（从右侧滑入）          │
└─────────────────────────────────────┘
     └─ 设置 Popover（点击弹出）
```

### 2. 设置面板：从常驻栏改为隐藏式 Popover

- **触发方式：** 点击"图谱设置"按钮
- **位置：** 桌面端在右上角 anchored popover，移动端在底部 bottom sheet
- **关闭方式：** 
  - 点击关闭按钮
  - 点击外部区域
  - 再次点击设置按钮
- **分组：** 节点 / 力学 / 显示三个区块
- **实时反馈：** 滑块旁边显示当前数值

### 3. 详情面板：从外挂栏改为图内滑出面板

- **触发方式：** 点击节点
- **位置：** 
  - 桌面端：从图谱右侧内部滑出，覆盖部分图谱区域
  - 移动端：从底部滑出为 bottom sheet
- **宽度：** 桌面端 `min(22rem, 40%)`，移动端全宽
- **动画：** 使用 `slide-in-right` / `slide-in-bottom` 动画
- **内容优化：**
  - 朋友节点：头像、名称、站点、简介、标签、关系、访问按钮
  - Tag 节点：标签名、关联朋友列表

### 4. 工具按钮优化

- **位置：** 卡片头部右侧
- **样式：** 轻量玻璃按钮，带 icon + 文字
- **移动端：** 只显示 icon，隐藏文字
- **按钮：** 重新聚焦 / 重置视图 / 图谱设置

### 5. 图例添加

- **位置：** 图谱画布左上角
- **内容：** 朋友节点示例 + 标签节点示例
- **样式：** 轻量玻璃胶囊，带 backdrop-filter

### 6. 视觉语言统一

- 使用现有主题变量（`--accent-color`, `--glass-bg`, `--fg-muted-*`）
- 玻璃感卡片 + 柔和阴影
- 主题色只用于 hover / active / 选中状态
- 圆角、边框、阴影与博客其他卡片保持一致
- 点阵背景更柔和（opacity 从 0.28 降至 0.22）

### 7. 响应式优化

**桌面端（> 960px）：**
- 详情面板从右侧滑入
- 设置 popover 在右上角

**移动端（≤ 960px）：**
- 详情面板从底部滑入，最高 70vh
- 设置 popover 从底部滑入，最高 80vh
- 工具按钮只显示 icon

**小屏（≤ 760px）：**
- 图谱高度降至 32rem
- 图例更紧凑

## 技术细节

### 详情渲染逻辑更新

- 朋友节点和 tag 节点分别渲染不同内容
- 朋友节点显示关系列表
- Tag 节点显示关联朋友列表
- 未选中节点时隐藏详情面板

### 设置面板交互

- 使用 `hidden` 属性控制显示 / 隐藏
- 监听 document click 实现点击外部关闭
- 支持 ESC 键关闭（浏览器默认行为）

### 控制输入反馈

- Range input 变化时同步更新 output 元素
- 数值格式化：小数保留 2 位，整数保留 0 位
